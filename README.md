# 📝 To-Do App

##  Live Demo

https://todo-app-zeta-pied-32.vercel.app

##  Features

* Add tasks
* Delete individual task
* Delete all tasks
* Mark tasks as completed
* Data saved using localStorage

## Tech Used

* HTML
* CSS
* JavaScript

##  Description

This is a simple To-Do application to manage daily tasks.
It stores data in browser using localStorage so tasks remain even after refresh.
